# ResearchGPT-Health

ResearchGPT-Health is a Gemini-powered healthcare research intelligence platform designed to analyze scientific literature, identify research gaps, generate executive reports, and enable grounded conversations across uploaded research papers.

Built with React, TypeScript, Vite, and Gemini AI.

---

## Features

### Literature Dashboard

* Research paper workspace management
* Clinical domain categorization
* Publication tracking
* Citation analytics
* Interactive research connectivity visualization

### Research Paper Upload

* Upload PDF, TXT, and Markdown research papers
* Metadata extraction
* Paper indexing
* Clinical literature organization

### AI Paper Summarization

* Automated abstract synthesis
* Key findings extraction
* Clinical interpretation
* Study objective identification
* Research significance analysis

### Research Gap Radar

* Methodological gap detection
* Study limitation analysis
* Outstanding question discovery
* Future research recommendations
* Priority scoring system

### Literature AI Chat

* Grounded conversations over indexed literature
* Multi-paper context awareness
* Evidence-based responses
* Cross-study comparisons
* Research Q&A assistant

### Executive Report Builder

* Multi-paper synthesis
* Clinical review generation
* Meta-analysis style summaries
* Executive reporting
* Publication-ready insights

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/TarunikaBalaji/ResearchGPT-Health.git
cd ResearchGPT-Health
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
APP_URL=http://localhost:3000
```

### 4. Run Development Server

```bash
npm run dev
```

Application:

```text
http://localhost:3000
```
